官方合作伙伴激励计划

第一章 计划概述

1.1 愿景与使命

Eazy Router TOKEN 生态合作计划致力于构建去中心化人工智能服务网络，通过创新的激励机制，推动 AI 能力的普惠化、民主化进程，让每一位参与者都能共享技术发展红利。

1.2 核心价值主张

  • 价值共创：参与者既是使用者，也是生态共建者
  • 收益共享：贡献越大，回报越丰厚
  • 生态共赢：多层级协同发展，构建可持续增长飞轮

1.3 合规声明

本计划严格遵守相关法律法规，所有激励均以合规方式进行。TOKEN 仅作为生态内价值流转媒介，不具备任何投资属性。

第三章 合作伙伴网络体系

3.1 合作伙伴等级架构

三级合作伙伴网络：

等级 称号 业绩门槛（TOKEN 激励业绩金额）
三级 生态共建者 ≥ 100 USD
二级 生态推动者 ≥ 5,000 USD
一级 生态引领者 ≥ 20,000 USD

3.2 网络拓展激励机制

3.2.1 双层发展模型

每个层级的合作伙伴可拓展两层网络：

  • 直接拓展的合作伙伴为一层网络节点
  • 一层节点拓展的合作伙伴为二层网络节点

3.2.2 网络业绩计量

合作伙伴业绩包含：

  1. 直接促成的 TOKEN 流转量
  2. 一层网络总流转量的 10%
  3. 二层网络总流转量的 5%

3.3 网络激励分配机制

3.3.1 激励分配公式

一层节点（直接拓展）：

生态引领者 A → 拓展 → 生态推动者 B（一层节点）
  → B 流转 TOKEN × 10% → A 激励

二层节点（间接拓展）：

生态引领者 A → 拓展 → 生态推动者 B → 拓展 → 生态共建者 C（二层节点）
  → C 流转 TOKEN × 5% → A 激励

3.3.2 分层激励标准

层级 激励比例
一层网络 流转量的 10%
二层网络 流转量的 5%
