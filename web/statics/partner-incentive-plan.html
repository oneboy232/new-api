第一章 计划概述Chapter 1 Program Overview

1.1 愿景与使命1.1 Vision & Mission

Eazy Router TOKEN 生态合作计划致力于构建去中心化人工智能服务网络，通过创新的激励机制，推动 AI 能力的普惠化、民主化进程，让每一位参与者都能共享技术发展红利。

The Eazy Router TOKEN Ecosystem Partnership Program is dedicated to building a decentralized AI service network. Through innovative incentive mechanisms, it promotes the democratization of AI capabilities, allowing every participant to share in the benefits of technological development.

1.2 核心价值主张1.2 Core Value Proposition

  • 价值共创：参与者既是使用者，也是生态共建者
  • Value Co-creation: Participants are both users and ecosystem builders
  • 收益共享：贡献越大，回报越丰厚
  • Revenue Sharing: Greater contribution brings greater rewards
  • 生态共赢：多层级协同发展，构建可持续增长飞轮
  • Ecosystem Win-Win: Multi-tier collaborative development building a sustainable growth flywheel

1.3 合规声明1.3 Compliance Statement

本计划严格遵守相关法律法规，所有激励均以合规方式进行。TOKEN 仅作为生态内价值流转媒介，不具备任何投资属性。

This program strictly complies with all applicable laws and regulations. All incentives are distributed in a compliant manner. TOKEN serves solely as a value transfer medium within the ecosystem and does not constitute an investment instrument.

3.2 网络拓展激励机制3.2 Network Expansion Incentive Mechanism

3.2.1 双层发展模型3.2.1 Two-Level Development Model

每个层级的合作伙伴可拓展两层网络：

Partners at each tier can expand two levels of network:

  • 直接拓展的合作伙伴为一层网络节点
  • Partners directly referred become Level-1 Network Nodes
  • 一层节点拓展的合作伙伴为二层网络节点
  • Partners referred by Level-1 Nodes become Level-2 Network Nodes

3.2.2 网络业绩计量3.2.2 Network Performance Measurement

合作伙伴业绩包含：

Partner performance includes:

  1. 直接促成的 TOKEN 流转量
  2. Directly facilitated TOKEN circulation volume
  3. 一层网络总流转量的 10%
  4. 10% of Level-1 network total circulation
  5. 二层网络总流转量的 5%
  6. 5% of Level-2 network total circulation

3.3 网络激励分配机制3.3 Network Incentive Distribution Mechanism

3.3.1 激励分配公式3.3.1 Incentive Distribution Formula

一层节点Level-1 Node（直接拓展Direct Referral）：

生态引领者 A → 拓展 → 生态推动者 B（一层节点）
  → B 流转 TOKEN × 10% → A 激励
Ecosystem Leader A → refers → Ecosystem Promoter B (Level-1 Node)
  → B's TOKEN circulation × 10% → Incentive to A

二层节点Level-2 Node（间接拓展Indirect Referral）：

生态引领者 A → 拓展 → 生态推动者 B → 拓展 → 生态共建者 C（二层节点）
  → C 流转 TOKEN × 5% → A 激励
Ecosystem Leader A → refers → Ecosystem Promoter B → refers → Ecosystem Builder C (Level-2 Node)
  → C's TOKEN circulation × 5% → Incentive to A

3.3.2 分层激励标准3.3.2 Tiered Incentive Standards

层级 Level 激励比例 Incentive Rate
一层网络 Level-1 Network 流转量的 10% of circulation volume 10%
二层网络 Level-2 Network 流转量的 5% of circulation volume 5%
