API 参考API Reference

Eazy Router 提供完整的 RESTful API 接口，分为 AI 模型接口 和 管理接口 两大类。Eazy Router provides a complete RESTful API, divided into AI Model APIs and Management APIs.

RESTful API 兼容 OpenAIOpenAI Compatible

AI 模型接口提供各种 AI 能力的调用，兼容 OpenAI API 格式，所有接口均使用 Bearer Token 鉴权。AI Models API provides access to various AI capabilities, compatible with OpenAI API format, all endpoints use Bearer Token authentication.

GET /v1/models 获取可用列表List available models

获取当前可用的列表，根据请求头自动识别返回格式（Claude/Gemini/OpenAI）。Get the list of available models, response format is auto-detected from request headers (Claude/Gemini/OpenAI).

注意：Note: 携带 x-api-key 返回原生 Claude 格式，携带 x-goog-api-key 返回原生 Gemini 格式，其他情况返回 OpenAI 格式。Send x-api-key for native Claude format, x-goog-api-key for native Gemini format, otherwise returns OpenAI format.
